loom.converter.DeserializeLuceneDocumentToStream
	-> Deserialises a stream stored in a Lucene document
	-> Does not evaluate any other fields in the document
	
loom.converter.SerializeStreamToLuceneDocument
	-> Uses Metamorph to transform the stream that creates the Lucene document
	-> Uses StreamToLuceneDocument to create the Lucene document
	-> Uses CGEntity as serialisation format
	
loom.converter.StreamToLuceneDocument
	-> Creates a Lucene document using a Lux DocBuilder
	
loom.converter.StringToQuery
	-> Parses a string into a Lucene query using lucenes default query parser

loom.sink.LuceneIndexWriter
	-> Adds Lucene documents to an index
	
loom.source.LuceneIndexSearcher
	-> Accepts a Lucene Query and emits a Lucene document for each match
	
search.pipe.LuceneDocDecoder
	-> Converts a Lucene document into a stream
	-> Emits a literal for each field
	-> If the document contains a serialised record, it is deserialised using CGEntityDecoder

search.index.BatchIndexer
	-> Builds Lucene documents and adds batches of them to the index
	-> Uses a custom interface does not integrate with Metastream
	
search.index.StreamIndexer
	-> StreamReceiver for indexing
	-> Uses a BatchIndexer internally
	
search.StreamIndexerBuilder
	-> Sets up a complete indexer configuration using StreamIndexer 
	
search.schema.LuceneDocBuilder
	-> Builds Lucene documents. Can be used to group different values into one Lucene document field.
